# Contributing to Awesome Agentic Coding

Thanks for helping improve **Awesome Agentic Coding**.

Our goal is simple: keep this list curated, useful, current, and biased toward tools that genuinely help teams build, run, or scale agentic coding workflows.

## What to Contribute

We welcome pull requests for:

- New tools that clearly fit an existing category
- Better official URLs
- Sharper, more accurate descriptions
- New categories when a meaningful ecosystem cluster emerges
- Broken link fixes and outdated project cleanups

## Entry Format

Please use this exact format for every item:

```md
* [Tool Name](URL) - A concise, 1-2 sentence description highlighting its specific value proposition.
```

## Submission Guidelines

Before opening a PR, please check that your contribution:

- Uses the project's **official** website or GitHub repository URL
- Has a clear open-source license; **CC0-1.0 is acceptable** when it clearly applies to the project
- Fits the category where you added it
- Adds specific value rather than generic hype
- Avoids duplicate entries
- Keeps descriptions concise and factual
- Matches the tone and formatting already used in the README

## Pull Request Tips

A great PR usually includes:

- The tool name
- Why it belongs in this list
- The category you selected
- Any evidence that the project is active, useful, or widely adopted

## Quality Bar

We prefer:

- Well-known or clearly differentiated tools
- Practical infrastructure and developer workflow value
- Accurate descriptions over marketing copy
- Curation over volume

## Local Rust Infrastructure Entries

The repository currently includes placeholder entries for local Rust-native infrastructure ideas such as **CAMP**, **GARC**, **WASP**, and **TICK**. If those projects later get their own repositories, websites, or documentation, PRs that upgrade the placeholders to real project links are very welcome.

## Be Kind

Please be respectful, constructive, and community-minded in issues and pull requests.
